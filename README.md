# Keybindings

- `Mouse wheel` : Zoom in / out
- `r` : Reset zoom
- `=` : Increase brush size
- `-` : Decrease brush size
- `0` : Reset brush size
- `Middle mouse` : Pan
- `w/s/a/d` : Pan up / down / left / right
- `Ctrl + z` : Undo
- ~~`Ctrl + y` : Redo~~
- `0-9` : Select color
- `Tab` : Hide / Show UI
